# ViewTune에 기여하기

버그 신고와 기능 제안은 GitHub Issues를 이용해 주세요. 재현 가능한 버그에는 대상 사이트, Chrome 버전, 영상 표시 모드, 실행한 단축키와 기대·실제 결과를 포함해 주세요.

코드 변경은 다음 원칙을 따릅니다.

1. 사이트 이름이나 고정 DOM ID보다 표준 `<video>`와 기능·기하 조건을 우선합니다.
2. 페이지의 기존 레이아웃과 플레이어 상태를 정확히 복원할 수 없는 변경은 적용하지 않습니다.
3. 새 권한이나 네트워크 요청은 필요성과 개인정보 영향을 먼저 설명합니다.
4. 변경 후 `npm run check`를 실행하고 관련 계약 테스트를 추가합니다.

Pull Request에는 변경 이유, 사용자 영향, 검증한 사이트와 화면 모드를 적어 주세요.
