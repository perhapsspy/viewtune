ViewTune

개인정보처리방침

시행일: 2026년 7월 14일

ViewTune은 웹 비디오의 재생 속도와 표시 방식을 제어하는 Chrome 확장입니다.

처리하는 정보

ViewTune은 기능을 수행하는 동안 현재 페이지의 HTML5 <video> 요소, 영상의 원본 크기, 재생 속도, 플레이어와 컨트롤의 배치 정보를 브라우저 메모리에서 일시적으로 확인합니다. 이 정보는 저장되지 않으며 개발자 또는 제3자 서버로 전송되지 않습니다.

사용자가 변경한 단축키와 화면 피드백 표시 여부는 chrome.storage.sync에 저장됩니다. Chrome 동기화가 활성화된 경우 Google Chrome이 같은 Google 계정의 기기 사이에서 이 설정을 동기화할 수 있습니다. ViewTune 개발자는 이 저장값에 접근하지 않습니다.

수집하지 않는 정보

ViewTune은 이름, 이메일 주소, 계정 정보, 인증 정보, 위치, 금융·건강 정보, 개인 통신, 방문 URL 기록, 영상 내용, 쿠키 또는 양식 입력값을 수집하지 않습니다.

네트워크 전송과 제3자 제공

ViewTune은 개발자 서버, 분석 서비스, 광고 네트워크를 사용하지 않습니다. 사용자 데이터를 판매하거나 제3자에게 제공하지 않습니다.

권한 사용

  • storage: 단축키와 화면 피드백 환경설정을 저장합니다.
  • 웹사이트 접근: 현재 페이지의 HTML5 비디오를 감지하고 사용자가 지정한 물리 키 단축키로 제어합니다. 접근한 페이지 정보는 기능 수행을 위한 로컬 처리 외의 목적으로 사용하지 않습니다.

보관과 삭제

페이지에서 일시 처리한 비디오·레이아웃 정보는 보관하지 않습니다. 저장된 환경설정은 사용자가 확장을 제거하거나 Chrome의 확장 데이터·동기화 데이터를 삭제하여 제거할 수 있습니다.

변경 사항

이 방침이 변경되면 시행일과 내용을 갱신하고 Chrome Web Store 또는 확장 UI에서 필요한 고지를 제공합니다.
