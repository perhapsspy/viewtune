# Chrome Web Store 목록 초안

## 제품 이름

ViewTune — Video Pace & Frame

## 짧은 설명

웹 비디오의 재생 속도와 화면 비율을 물리 키 단축키로 가볍게 제어합니다.

## 카테고리와 언어

- 기본 카테고리: 생산성(Productivity)
- 기본 언어: 한국어
- 배포 범위: Public
- 지역: 전체 지역

## 상세 설명

ViewTune은 YouTube를 포함한 HTML5 웹 비디오의 재생 속도와 화면을 키보드로 제어하는 가벼운 Chrome 확장입니다.

평소에는 페이지 위에 상시 버튼을 띄우지 않습니다. 물리 키 단축키 또는 확장 팝업으로 조작하며, 성공한 경우에만 영상 위에 짧은 피드백을 표시합니다.

주요 기능

- 재생 속도를 0.5× 단위로 낮추거나 높이고 1×로 초기화
- 플레이어와 컨트롤을 현재 브라우저 탭에 맞추는 창 맞춤
- 현재 플레이어 또는 전체화면 크기를 유지하면서 영상만 확대하는 21:9 채우기
- 한글/영문 입력 상태와 무관한 물리 키 위치 기반 단축키
- 모든 단축키 변경과 화면 피드백 설정

기본 단축키

- U: 21:9 확대 토글
- V: 창 맞춤 토글
- [: 속도 0.5× 낮춤
- ]: 속도 0.5× 높임
- R: 속도 1× 초기화

ViewTune은 광고, 분석, 계정 기능 또는 개발자 서버를 사용하지 않습니다. 현재 페이지의 비디오와 플레이어 구조는 기능 수행 중 브라우저 안에서만 처리하며 저장하거나 외부로 전송하지 않습니다. 단축키와 화면 피드백 설정만 Chrome Storage Sync에 저장됩니다.

제한 사항

- HTML `<video>` 요소를 노출하지 않는 DRM 또는 canvas 기반 플레이어는 제어할 수 없습니다.
- 사이트가 안전한 플레이어 경계를 제공하지 않거나 native fullscreen·Picture-in-Picture와 충돌할 때 창 맞춤은 페이지를 변경하지 않고 거절합니다.
- 21:9 확대는 현재 화면이 원본 영상보다 충분히 넓을 때만 적용되며 위아래 일부가 잘릴 수 있습니다.
