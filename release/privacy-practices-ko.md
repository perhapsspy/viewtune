# Privacy practices 입력 초안

## 단일 목적

ViewTune은 사용자가 현재 보고 있는 HTML5 웹 비디오의 재생 속도와 표시 방식을 물리 키 단축키와 확장 팝업으로 제어하게 합니다.

## `storage` 권한 사유

사용자가 지정한 물리 키 단축키와 화면 피드백 표시 여부를 `chrome.storage.sync`에 저장하기 위해 필요합니다. 개발자 서버로 전송하지 않으며 개발자는 저장값에 접근하지 않습니다. 사용자가 Chrome 동기화를 사용하면 Google Chrome이 같은 계정의 기기 사이에서 설정을 동기화할 수 있습니다.

## 사이트 접근 사유

정적 content script가 HTTP/HTTPS 페이지의 HTML5 `<video>` 요소와 플레이어 경계를 감지하고, 사용자가 확장 아이콘을 먼저 누르지 않아도 물리 키 단축키를 사용할 수 있게 하려면 모든 일반 웹사이트에서 실행되어야 합니다. ViewTune은 URL, 페이지 본문, 영상 내용, 쿠키, 양식, 계정 정보를 저장하거나 개발자에게 전송하지 않습니다.

`host_permissions`는 별도로 요청하지 않습니다. 사이트 실행 범위는 `content_scripts.matches`의 `http://*/*`, `https://*/*`에만 선언합니다.

## 원격 코드

- 선택: 원격 코드를 사용하지 않음
- 설명: 모든 JavaScript와 CSS는 업로드 ZIP 안에 포함됩니다. 외부 스크립트, 원격 설정, `eval` 또는 동적 코드 다운로드를 사용하지 않습니다.

## 데이터 사용 신고

- `웹사이트 콘텐츠`: 선택. 현재 페이지의 `<video>` 요소·플레이어 DOM 구조·기하 정보를 기능 수행 중 로컬 메모리에서 일시 처리합니다.
- `사용자 활동`: 선택. `keydown` 이벤트에서 설정된 ViewTune 단축키와 일치하는 물리 키 위치인지 로컬에서 비교하며, 일반 키 입력이나 입력 텍스트를 기록하지 않습니다.
- 단축키와 피드백 환경설정은 Chrome Storage Sync에 보관되지만 개발자에게 전달되지 않습니다.
- 개인 식별 정보, 건강·금융·인증 정보, 개인 통신, 위치, 웹 탐색 기록은 수집하지 않습니다.

## Limited Use 확인

실제 체크박스 문구가 아래 내용과 일치하는지 확인한 뒤 모두 인증합니다.

- 사용자 대면 단일 목적 제공에 필요한 범위로만 데이터를 사용
- 데이터를 판매하거나 광고·신용·대출 목적으로 사용하지 않음
- 허용된 목적 외 제3자 전송 없음
- 사람이 사용자 데이터를 읽도록 허용하지 않음

## 개인정보처리방침 URL

`privacy-policy.html`을 공개 HTTPS 주소에 게시한 뒤 그 URL을 입력합니다. 로컬 파일 경로나 비공개 저장소 URL은 사용할 수 없습니다.
