# 심사자 테스트 절차

로그인이나 테스트 계정은 필요하지 않습니다.

1. 확장을 설치한 뒤 공개 HTML5 비디오가 있는 페이지를 엽니다. YouTube 공개 영상 또는 `https://interactive-examples.mdn.mozilla.net/pages/tabbed/video.html`을 사용할 수 있습니다.
2. 비디오를 재생하고 `[`와 `]`를 눌러 재생 속도가 0.5× 단위로 변경되는지 확인합니다.
3. `R`을 눌러 속도가 1×로 돌아오는지 확인합니다.
4. 안전한 HTML 플레이어 경계와 가시 컨트롤이 있는 페이지에서 `V`를 눌러 플레이어와 컨트롤이 탭 화면을 채우고, 다시 눌러 원래 상태로 복원되는지 확인합니다.
5. 원본 영상보다 넓은 플레이어 또는 울트라와이드 전체화면에서 `U`를 눌러 현재 프레임 크기는 유지한 채 영상만 빈 영역 없이 확대되는지 확인합니다. 일반 16:9 프레임에서는 적용 조건이 없다는 안내가 정상입니다.
6. 확장 팝업에서 같은 기능을 버튼으로 실행하고 설정 버튼을 열어 각 물리 키 단축키를 변경할 수 있는지 확인합니다.

안전상 ViewTune은 iframe 내부에서 탭 전체 창 맞춤을 시도하거나, native fullscreen·Picture-in-Picture·기존 Popover와 충돌하거나, 플레이어 경계를 확실히 찾을 수 없는 경우 V를 적용하지 않습니다. 이는 의도된 동작입니다.
