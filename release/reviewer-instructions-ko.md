# 심사자 테스트 절차

로그인이나 테스트 계정은 필요하지 않습니다.

1. 확장을 설치한 뒤 공개 HTML5 비디오가 있는 페이지를 엽니다. YouTube 공개 영상 또는 `https://interactive-examples.mdn.mozilla.net/pages/tabbed/video.html`을 사용할 수 있습니다.
2. 비디오를 재생하고 `[`와 `]`를 눌러 재생 속도가 0.5× 단위로 변경되는지 확인합니다.
3. `R`을 눌러 속도가 1×로 돌아오는지 확인합니다.
4. `G`를 눌러 기본 목표 속도 2×가 바로 적용되는지 확인합니다.
5. 안전한 HTML 플레이어 경계와 컨트롤이 있는 페이지에서 `V`를 눌러 플레이어와 컨트롤이 탭 화면을 채우는지 확인합니다. `V`를 다시 누르거나 `Esc`를 누르면 원래 상태로 복원됩니다.
6. 원본 영상보다 넓은 플레이어 또는 울트라와이드 전체화면에서 `B`를 눌러 현재 프레임 크기는 유지한 채 영상만 빈 영역 없이 확대되는지 확인합니다. 일반 16:9 프레임에서는 적용 조건이 없다는 안내가 정상입니다.
7. 확장 팝업에서 같은 기능을 버튼으로 실행하고 톱니 버튼을 누릅니다. 팝업 안에서 각 물리 키 단축키, 목표 속도와 화면 피드백을 변경할 수 있습니다.
8. Chrome 표시 언어가 한국어이면 한국어 UI, 그 밖의 지원 환경에서는 영어 UI가 표시됩니다.

선택적인 호환성 확인

- YouTube에서 일반 모드와 `T` 영화관 모드를 오가며 `V`를 여러 번 켜고 꺼도 플레이어와 컨트롤이 함께 복원됩니다.
- 라프텔에서 컨트롤이 자동으로 숨겨진 뒤에도 `V`가 확인된 플레이어 프레임을 탭 화면에 맞춥니다. 이 확인에는 서비스 이용 권한이 있는 계정이 필요할 수 있습니다.

안전상 ViewTune은 iframe 내부에서 탭 전체 창 맞춤을 시도하거나, native fullscreen·Picture-in-Picture·기존 Popover와 충돌하거나, 플레이어 경계를 확실히 찾을 수 없는 경우 V를 적용하지 않습니다. 이는 의도된 동작입니다.
